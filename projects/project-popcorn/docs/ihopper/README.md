# ihopper 프로젝트 자료

> 의안 데이터 파이프라인 메인 프로젝트 참고 자료

## 개요

ihopper는 국회 의안 데이터를 수집/변환/분석하는 메인 파이프라인 프로젝트.
본 폴더는 지식그래프 구축을 위해 공유받은 참고 자료를 보관.

## 아키텍처 (담당 영역)

```
                                ┌→ OpenSearch (텍스트 검색)
ods_core (정규화) → dm (비정규화) ─┤
                                └→ Knowledge Graph (관계 분석) ← 담당
```

## 지식그래프 범위

| 엔티티 | 설명 |
|--------|------|
| 의안 (Bill) | 법률안, 결의안 등 |
| 의원 (Member) | 국회의원 |
| 위원회 (Committee) | 상임위, 특위 |
| 법률 (Law) | 현행 법률 |
| 조문 (Article) | 법률 조항 |
